# Encrypted Messaging App (End-to-End)

## What i build:
A small chat app where:

- Each user has their own key pair
- Messages are encrypted client-side
- Server stores only ciphertext
- Optional: message signing (integrity)

## Skills covered:

- Asymmetric cryptography (RSA, ECC)
- Symmetric encryption (AES)
- Key exchange (Diffie-Hellman)
- Secure key storage

## Tech:
React + Node
